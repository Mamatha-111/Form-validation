Git Commands

  • git init

    this comand is used to start a repository while creating a new repository.

  • git add

    this comand is used to add the file to the staging area.

  • git commit -m"comments"

    this comand is used to record the version history of a file.

  • git commit -a

    this comand commits the files you have added with add command and also commits files in which chnages are done.

  • git diff

    this comand is used to show the differences which are not staged.

  • git status

    this comand shows the files in that folder that have to be commited

  • git branch

    this comand shows all the local branches in current repository.

  • git checkout [branch name]

    this comand is used to switch between branches.

  • git merge [branch name]

    this comand is used to merge the specified branch history to current branch.

  • git push

    this comand is used to send the file to remote repository.

  • git pull

    this comand is used to fetch and merge changes on remote server to your working directory.

  • git clone

    this comand is used to copy files from remote repository to local repository.

  • git reset

    this comand is used to reset the files to last committed state.

  • git log

    this comand shows all the past commits
